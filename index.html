Meus Projetos

Projeto - On Firer

On Firer é uma loja de roupas cristãs desenvolvida como uma aplicação web de e-commerce. O projeto oferece uma experiência de compra intuitiva e eficiente, destacando duas coleções principais da marca: uma blusa de frio e uma camiseta. A aplicação permite aos usuários visualizar todos os produtos disponíveis na página inicial e realizar pesquisas por palavras-chave relacionadas aos produtos. Os resultados da pesquisa são filtrados com base em título, descrição e tags, e cada item oferece links diretos para mais informações, incluindo páginas específicas dos produtos e o Instagram da marca. O projeto utiliza uma combinação de HTML, CSS e JavaScript para criar uma interface amigável e funcional. A estrutura HTML define a organização da página, incluindo um cabeçalho, um campo de pesquisa e uma seção para exibir os produtos. O CSS é responsável pela estilização visual, aplicando um layout responsivo com Flexbox e media queries, além de um tema coeso com imagens de fundo e gradientes. A lógica da aplicação é gerida pelo JavaScript, que inclui funções para exibir todos os produtos e realizar pesquisas, além de manipular o DOM para atualizar a página com os resultados. Os arquivos principais do projeto são index.html, que contém a estrutura da aplicação; style.css, que define a aparência e o layout; app.js, que contém o código JavaScript para exibição e pesquisa; e dados.js, que armazena os dados dos produtos, como títulos, descrições, links e tags. O On Firer é um exemplo de como combinar design e desenvolvimento para criar uma experiência de usuário envolvente e eficaz.
